# Metrea Mission Data Limited

> Systems engineering, Tactical Data Link software, EW data analytics and space consultancy for the defence and security sectors.

## What they do

Metrea Mission Data delivers systems engineering, Tactical Data Link software products and technical services, Electronic Warfare data analytics, space and information assurance consultancy. The company's subject matter experts cover ISTAR, Tactical Data Links, communications and information assurance, supporting defence acquisition and capability development.

## Products and solutions

- **Tactical Data Link software** - Software products and technical services for TDL integration and data exchange.
- **EW data analytics** - Electronic warfare data analysis and advisory.
- **Space consultancy** - Space sector advisory and consultancy services.
- **Information assurance** - Security and information assurance consultancy across defence programmes.
- **Simulation and modelling** - Mission data and simulation services.
