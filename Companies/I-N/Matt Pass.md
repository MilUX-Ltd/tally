# Matt Pass

> London-based cyber security and web application development consultancy serving security, defence and government clients.

## What they do

Matt Pass Ltd provides web application development, cyber security and data processing services, with a focus on security and defence sector clients including UK and US government bodies. The company states it has provided solutions since 2015 and has worked professionally since 1999, delivering around 450 projects for over 300 clients. It has delivered cyber security training to over 250,000 users online and in person, with the aim of developing future cyber security professionals.

## Products and solutions

- **Web application development** - full-stack development covering frontend, backend, DevOps and databases.
- **Cyber security services** - cyber security expertise for security and defence sector clients.
- **Cyber security training** - instruction delivered online and in person to build cyber security skills.

## Frameworks, accreditations and memberships

- **Accreditations:** GitHub Advanced Security (GHAS) accredited.
- **Memberships:** ADS Group, UK Cyber Security Council corporate member.
