# Mollis Group

> Multi-cloud security consultancy helping organisations operate securely in cloud environments.

## What they do

Mollis Group is a multi-cloud security consultancy that helps organisations operate securely in the cloud, providing security guidance it describes as simple, consistent and transparent. The company organises its work around three areas: building security into development (DevSecOps), assuring systems through testing and framework compliance, and operating secure multi-cloud environments on an ongoing basis.

## Products and solutions

- **Penetration testing** - ethical hacking services, from one-off tests to embedded specialists working continuously within agile teams.
- **Threat hunting** - continuous monitoring for threats across client environments.
- **Incident response** - forensics, logging and monitoring support following a security incident.
- **Vulnerability assessment (Swordfish)** - vulnerability assessment as a service.
- **Security architecture services** - design work covering identity and access management, data encryption and network segmentation.
- **TSCM sweeps (Cyber Sweep)** - technical surveillance countermeasures using government-accredited equipment, alongside wireless and RF assurance.
- **OSINT and dark web investigations** - open-source and dark web intelligence gathering.
- **Governance, risk and compliance advisory** - cloud governance, risk management and compliance support, including cloud financial governance and fractional security leadership (vCISO/vCTO-style advisory).
