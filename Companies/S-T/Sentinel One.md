# Sentinel One

> Cybersecurity platform provider delivering autonomous endpoint protection, detection and response, IoT security, and cloud workload protection through a unified data lake.

## What they do

SentinelOne's Singularity platform unifies endpoint protection (EPP), endpoint detection and response (EDR), IoT security, and cloud workload protection (CWPP) into a single platform backed by a centralised data lake. The platform provides real-time, autonomous threat detection and response across enterprise assets without requiring manual intervention.

## Products and solutions

- **Singularity Platform** - Unified security data lake integrating EPP, EDR, IoT security, and CWPP with autonomous response capability.
