# Google Cloud Public Sector UK Ltd

> UK entity established for contracting to HMG, providing cloud, AI and cybersecurity capabilities to UK public sector and defence organisations.

## What they do

Google Cloud Public Sector UK Ltd is the UK contracting entity through which Google delivers cloud and AI capabilities to HMG customers. It provides a cloud-native platform designed for handling sensitive government data, including frontline intelligence and incident response for security operations, AI-powered cloud-native tools for security operations centres, and a secure-by-design, secure-by-default cloud infrastructure. The entity supports public sector organisations across central government, defence and national security.

## Products and solutions

- **Secure cloud platform** - cloud infrastructure built for UK government data handling requirements.
- **AI-powered security operations** - tools for security operations centres covering threat detection and incident response.
- **Sensitive data protection** - capabilities for managing and protecting classified and sensitive government data.
