# FlexSR
> UK startup developing small-model, low-code speech recognition delivered as an API-driven microservice.
## What they do
FlexSR develops universal speech-recognition technology built for integration into demanding environments. The core product is a lightweight model deployable as a containerised microservice, designed for solution developers to embed voice input into applications without large infrastructure overhead. The D3IP directory description notes positioning for real-time, location-aware use cases in defence and security contexts.
