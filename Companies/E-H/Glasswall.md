# Glasswall

> Cybersecurity company providing Content Disarm and Reconstruction (CDR) technology to protect government and defence organisations from file-based threats.

## What they do

Glasswall Solutions Limited develops zero-trust file security technology based on Content Disarm and Reconstruction (CDR). Its products treat every file as untrusted, validating, rebuilding, and cleaning each file to a known-good, malware-free standard before it reaches a user or system. The company's technology is deployed by government agencies and defence organisations across the Five Eyes nations, and has been adopted for cross-domain solutions, secure file transfer portals, and disconnected, degraded, intermittent, and limited (DDIL) environments. In April 2024, Glasswall joined the D3IP collaboration community, which provides access to the UK Defence BattleLab's co-creation space and the NSTIx innovation programme. The company is backed by PSG Equity.

## Products and solutions

- **Glasswall Halo (Server)** - Scalable CDR server solution deployable on-premises, in the cloud, or in air-gapped networks.
- **Glasswall Embedded Engine (SDK)** - Software development kit enabling CDR to be integrated into third-party applications and services.
- **Glasswall Meteor (Desktop)** - Automated desktop application for processing local and cloud storage files.
- **Glasswall Foresight** - AI-powered threat prediction capability using machine learning trained on file structural telemetry to identify unknown and zero-day malware.
