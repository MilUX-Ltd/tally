# Huntsman Security

> Cyber security analytics company specialising in real-time threat detection, security log management, and behavioural anomaly detection.

## What they do

Huntsman delivers security analytics and intelligent security log, event and incident management solutions that analyse and correlate events in real time to provide early detection of security threats and regulatory compliance issues. The company's patented Behavioural Anomaly Detection technology automatically baselines and processes millions of log records to detect malicious activity in real time, prioritising security and compliance issues through a single-console view.

## Products and solutions

- **Huntsman SIEM** - Security information and event management platform providing real-time correlation, alerting and drilldown dashboards.
- **Behavioural Anomaly Detection** - Patented technology that baselines user and system behaviour to identify malicious activity automatically.
