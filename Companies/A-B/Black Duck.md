# Black Duck

> Application security testing and software composition analysis provider.

## What they do

Black Duck provides application security testing and software composition analysis, helping organisations find vulnerabilities, manage open-source licensing risk and secure software supply chains. It serves regulated sectors including aerospace, defence and critical infrastructure, and supports compliance regimes such as software bills of materials. Formerly the Synopsys Software Integrity Group, it became an independent company in 2024 and is headquartered in Burlington, Massachusetts.

## Products and solutions

- **Polaris Platform** - cloud-native application security testing combining static analysis, software composition analysis and dynamic analysis.
- **Coverity** - static application security testing across many programming languages.
- **Black Duck SCA** - software composition analysis for open-source visibility and risk.
