# Prizsm Technologies

> UK data-security firm providing quantum-resistant computational storage that fragments and scatters data across multiple clouds.

## What they do

Prizsm describes itself as a computational storage capability providing resilient, quantum-resistant, hyper-secure cloud storage and communications. Its algorithm disrupts data at bit-level and randomly distributes redundant, complementary binary digits across multiple cloud platforms, so information is stored, in its own words, everywhere and nowhere. By sitting between users and cloud storage platforms, it aims to let organisations use cheap, elastic cloud services in an inherently secure way, with data recoverable even if a cloud provider is lost.

## Products and solutions

- **Prizsm Storage** - core store-and-retrieve product using bit-level data disaggregation and multi-cloud distribution for resilient, quantum-resistant storage.
- **Prizsm Edge** - secure store-and-forward communication for edge computing and IoT, allowing multiple independent channels to share cloud storage without crosstalk.
- **Prizsm Key** - secure key sharing built on the Prizsm algorithm.
