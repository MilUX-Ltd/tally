# Planned Link Ltd

> Cyber security and network consultancy providing technical security solutions to defence and government organisations.

## What they do

Planned Link delivers technical security consultancy across networking, infrastructure, and cloud platforms, with a focus on vulnerability scanning, security architecture, and engineering. The team has experience in Analysis, Engineering, and Architecture across both Security and Networking disciplines. The company describes its security services as including advice, guidance and implementation of firewalls, vulnerability management, analytics and identity access management, alongside Microsoft licensing and cloud platform support (Microsoft Azure, Microsoft 365 and Dynamics 365) as a CSP partner.

## Products and solutions

- **Vulnerability scanning and management** - implementation of vulnerability scanning, or a fully managed vulnerability scanning service, shaped to the client's need and budget. The managed service ingests and normalises vulnerability data across an enterprise, feeding dashboards, reports and risk frameworks drawn from infrastructure, application and penetration-test tooling.
- **Microsoft licensing and cloud services** - support across Microsoft Azure, Microsoft 365 and Dynamics 365 as a Microsoft CSP partner, from single licences to large-scale estates.
- **IT security services** - advice, guidance and implementation of firewalls, vulnerability management, analytics and identity access management.

## Frameworks, accreditations and memberships

- **Memberships:** ADS Group
- **Accreditations:** ISO 9001, ISO 27001:2022, Cyber Essentials Plus, JOSCAR Registered, Defence Cyber Certification (DCC) Lv0–Lv3
