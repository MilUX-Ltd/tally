# Crackerjack Technologies

> UK software and cyber security company developing applications and digital services for technical users in defence and related sectors.

## What they do

Crackerjack Technologies Ltd builds software applications for technical users, with capabilities spanning cyber security, data analysis, software modelling, and digital services. The company holds Cyber Essentials Plus certification and is an ADS Group member, positioning it within the aerospace, defence, security and space supply chain.

## Products and solutions

- **Software applications** - purpose-built applications for technical users, described as modern and intuitive.
- **Cyber security and systems integrity services** - cyber security support and systems assurance.
- **Data analysis and software modelling** - analytical and modelling software services.
- **Digital services** - web and digital solutions.

## Frameworks, accreditations and memberships

- **Accreditations:** Cyber Essentials, Cyber Essentials Plus
- **Memberships:** ADS Group
