# Drop Table Ltd

> Microsoft-certified Azure cloud security consultancy serving highly regulated industries including defence.

## What they do

Drop Table designs, builds, and secures Azure cloud platforms for organisations in financial services, healthcare, manufacturing, and defence. The company offers services across cloud architecture, security engineering, DevSecOps, platform engineering, compliance, and automated security assessments. It holds Microsoft Solutions Partner status with advanced specialisations in cloud security.

## Products and solutions

- **Drop Table Pulse** - An automated security assessment tool for Azure DevOps. Runs 150+ security checks across pipelines, repositories, and service connections and produces compliance reporting and remediation guidance.
- **Azure DevOps Security Assessment** - A structured assessment service identifying misconfigurations, security risks, and compliance gaps in Azure DevOps environments.

## Defence work

Drop Table has documented defence sector delivery, including a Zero Trust security architecture designed and implemented for a major defence contractor to enable secure collaboration across classified and unclassified environments. The company lists ISO 27001, SOC 2, and DORA compliance services alongside defence-relevant security engineering.
