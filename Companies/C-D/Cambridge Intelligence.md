# Cambridge Intelligence Limited

> Cambridge-based developer of graph, timeline and geospatial data-visualisation SDKs used to build investigation tools for mission-critical software.

## What they do

Cambridge Intelligence provides visualisation software development kits (SDKs) that developers and product teams embed in their own applications to enable interactive exploration of complex, connected data. The tools are used in high-stakes domains including law enforcement, national security, cybersecurity and fraud prevention, where clarity, performance and reliability matter. The stated mission is to help people make sense of complex, connected data.

## Products and solutions

- **KeyLines** - JavaScript SDK for graph and network visualisation.
- **ReGraph** - React SDK for state-driven graph visualisation.
- **KronoGraph** - timeline visualisation SDK for investigating event data at scale.
- **MapWeave** - geospatial visualisation SDK for uncovering connections in map-based data.
