# Dynamic Devices Ltd

> Liverpool embedded systems consultancy developing satellite-enabled casualty tracking and hardware security technology for dual-use defence and emergency services applications.

## What they do

Dynamic Devices is an embedded systems consultancy with over two decades of experience in embedded Linux, RTOS, and IoT integration. Its most defence-relevant work is INST (Instantly Networked Smart Triage), an ESA-funded satellite-enabled casualty tracking system that uses low-cost IoT tags and a multi-bearer mesh network, including satellite links, to give commanders real-time visibility of casualty location, number, and triage urgency. The company also contributed to the UKRI Digital Security by Design programme, integrating CHERI/Morello memory-safe processor architecture into the Yocto Embedded Linux build system.

## Products and solutions

- **INST (Instantly Networked Smart Triage)** - Satellite-enabled mass casualty tracking system; IoT tags transmit triage data via mesh and satellite bearers to a digital-twin command map. Site Acceptance Test completed February 2026.
- **meta-cheri (Yocto layer)** - Open-source Yocto/OpenEmbedded layer adding CHERI/Morello build support for hardware-enforced memory safety; contributed to the UKRI DSbD Technology Access Programme.
