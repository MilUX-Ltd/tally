# Cunning Running Software Ltd

> Geospatial mapping, visualisation and planning software for defence, counter-terrorism and security planning.

## What they do

Cunning Running provides specialist 2D and 3D mapping, visualisation and planning software that supports decision-making in mission-critical operations. The company specialises in using GIS to analyse, visualise and understand threats and mitigations, supporting force protection, counter-terrorism, search and law-enforcement tasks. Alongside packaged software it provides training, consultancy and integration. The company moved to an Employee Ownership Trust in 2022.

## Products and solutions

- **FPPT (Force Protection Planning Tool)** - military application supporting force-protection planning.
- **CTS-PRAS (Counter Terrorism and Search - Position Ranking Analysis System)** - civilian application for counter-terrorism and search tasks.
- **SAM-PRAS** - solution used at airports and airbases for security and risk analysis.
- **PRAS family** - tools supporting search (missing persons, forensics, counter-terrorism) and unmanned aerial vehicle tasking.
