# Corelight UK Ltd

> UK entity of Corelight, a network detection and response (NDR) and threat-hunting platform vendor.

## What they do

Corelight provides an open network detection and response (NDR) platform built on the open-source Zeek project. It turns network and cloud activity into evidence to power threat detection, hunting and incident response, identifying adversarial techniques mapped across the MITRE ATT&CK framework using AI, machine learning, behavioural analytics, curated signatures and threat intelligence. Corelight UK Ltd is the UK-registered entity; the group is headquartered in San Francisco, where the parent company was founded in 2013.

## Products and solutions

- **Open NDR platform** - network detection and response built on Zeek, transforming network traffic into security evidence.
- **Threat detection and analytics** - multi-layered detection combining AI, behavioural analytics, curated signatures and threat intelligence.
- **Lateral movement detection** - identification of suspicious internal access patterns and Kerberos attacks.
