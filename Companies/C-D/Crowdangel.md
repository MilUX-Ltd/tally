# Crowdangel

> Situational messaging and crowd-safety platform supporting Martyn's Law compliance at live events.

## What they do

Crowd Angel provides a situational messaging platform that helps event organisers and venues assess threats, monitor crowd flow and communicate with attendees during incidents. The platform supports compliance with Martyn's Law, the UK's counter-terrorism protective security legislation for public venues. It is delivered as a web portal, an embeddable API and a control-room dashboard.

## Products and solutions

- **Web portal** - real-time crowd monitoring with density and flow visualisation.
- **Embeddable API** - integration into existing event apps.
- **Control room** - dashboard for planning, monitoring and incident response.
