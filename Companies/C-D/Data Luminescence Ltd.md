# Data Luminescence Ltd

> Energy-efficient embedded database software, with defence and sensor-network applications.

## What they do

Data Luminescence develops energy-efficient, embedded data-management software designed to cut energy consumption and improve performance over legacy systems. Its products target defence and industrial sensor networks and Internet-of-Things applications, with a security framework offering encryption for military-grade use. The company is based at the Harwell Science and Innovation Campus.

## Products and solutions

- **Atomics** - memory allocation toolkit for C and C++ developers.
- **Core** - lightweight embedded database for sensor networks and IoT.
- **Ultra** - service embedding Core with a security and encryption framework.
